## 실행 방법
1.  package.json에 정의되어 있는 패키지 설치 (최초 1회)
    ```shell
    $ yarn
    ```
1.  개발 모드로 서버 실행
    ```shell
    $ yarn dev
    ```
1.  프로젝트 빌드
    ```shell
    $ yarn build
    ```
1.  운영 모드로 서버 실행
    ```shell
    $ yarn start
    ```
